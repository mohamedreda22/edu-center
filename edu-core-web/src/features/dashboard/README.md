# Dashboard Module
أكاديمية ركان - لوحة التحكم

## Overview
This module provides a visual overview of the institute's operations. The dashboard is role-based, ensuring that users only see data relevant to their permissions.

### Role-Based Access
- `ADMIN`: Full access to students, teachers, lessons, revenue metrics, and recent activity logs.
- `ACCOUNTANT`: Access to financial metrics and student/teacher counts.
- `RECEPTIONIST`: Access to student and lesson counts only.
- `TEACHER`: Professional dashboard with personal schedule and performance stats.

## Status
The dashboard is fully integrated with:
1. **Lessons Data:** Displays upcoming schedules for teachers and active lesson counts for staff.
2. **Payroll Data:** Revenue metrics use real-time aggregation from completed lessons.
3. **Activity Logs:** The "Recent Activity" feed shows the latest system operations to administrators.

## Features
- Real-time statistics (StatCards).
- Personalized greetings.
- Smart scheduling widgets.
- Activity feed for monitoring.
- Fully localized in Arabic.
